# Tetris Game

## 서비스 소개

![](https://i.imgur.com/JaeC25n.png)


### 게임 기능

* 시작
* 일시정지
* 다시 시작
* 방향 이동 -> 양쪽 방향키
* 빠르게 내리기 -> 스페이스바
* 회전 -> 윗쪽 방향키
* 점수판
* 다음 블락 미리보기

### 게임 시연 동영상

## 개발적 특징
* 회전변환을 이용한 회전 구현
* View가 Model의 변화를 구독하는 Observe Pattern 경험
* 새로운 Block 모양을 쉽게 추가할 수 있도록 Model 설계
